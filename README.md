# Tiny Config
A very basic configuration API that is used by Proton.

## How to use:

### Import as a dependency:
Gradle Groovy:
```groovy
repositories {
    maven { url = "https://nexus.dyonb.nl/repository/proton/" }
}

// ...

dependencies {
    implementation("io.github.protonmc:tiny_config:version")
}
```
